# Manifolds
Def : Let $X$ be a set. A topology $\tau$ on $X$ is a collection of subsets of $X$, i.e., $\tau \subset \mathcal{P}(X)$, satisfying the following: <br>
    1. $X,\phi \in \tau$ <br>
    2. $\\{G_\alpha:\alpha \in \Lambda \\} \subset \tau \implies \bigcup_{\alpha \in \Lambda} G_\alpha \in \tau$ <br>
    3. $G_1,G_2,...,G_n \in \tau \implies \bigcap_{j=i}^n G_j \in \tau$ <br>
Def : A topological space is pair of object $(X,\tau)$ where $X$ is a set and $\tau$ is a topology on $X$. <br>
Def : If $(X,\tau)$ is a topological space then sets in $\tau$ are called open sets in the given topological space. <br>
